Features

  • Multiple output formats: Interactive HTML or static PNG/SVG
  • Automatic route generation from chronological dates
  • Trip grouping based on date proximity
  • Routes from home with return arcs
  • Correct handling of international dateline crossings
  • Import locations from geotagged photos
  • Iframe wrappers for easy website embedding
  • Simple YAML configuration format
